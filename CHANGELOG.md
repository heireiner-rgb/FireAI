# Changelog

## 0.1.0 - 2026-03-02

### Added
- Scenario core with load/save JSON support.
- Session engine with rule-based dialogue and transcript export.
- In-memory service layer for scenarios and sessions.
- HTTP API endpoints for scenarios, sessions and audio settings/testing.
- Web UI for scenario management, audio settings, and message simulation.
- CLI prototype for terminal-based interaction.
- Unit/integration tests for session, service and API workflows.
- Release helper files (`Makefile`, `requirements.txt`, `LICENSE`).
